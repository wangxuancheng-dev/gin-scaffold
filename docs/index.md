# gin-scaffold 文档中心

企业级 Go 脚手架（Gin + GORM + Redis + Asynq + Swagger + 可观测性）。

适用定位：**中小团队可生产使用**，强调稳定、可维护、可观测与快速迭代。

## 你可以从这里开始

- 新同学：[新人入门（架构 + FAQ）](/guide/onboarding) -> `快速开始` -> `配置说明` -> `命令系统`
- 运维同学：`生产运行手册` -> `上线检查清单`
- 后端同学：`定时任务中心`、`日志与可观测`
- 按角色走：`开发同学` / `运维同学` / `测试同学` 阅读路径
